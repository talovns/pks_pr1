# Правила пользования калькулятором:
- Для запуска программы запустите файл Program.cs 
- После откроется терминал, в котором вы вводите свое выражение, следуя инструкциям на экране
- Для получения ответа нажмите на клавиатуре ENTER

# Ограничения : 
- Используйте числа разрядностью от -10 ^ 9 до 10 ^ 9
- Не делить на ноль (хотя ошибка выведется на экран)

# Возможные ошибки :
- Некорректный ввод знаков или чисел приведет к ошибке (если в случае знака будет предупреждение, то некорректный йввод числа сломает программу)


## Описание доступных знаков 
| Название | Описание                                                      |
|----------|---------------------------------------------------------------|
| x + y    | Сложение - возвращает сумму чисел x и y                       |
| x - y    | Вычитание - возвращает разность чисел x и y                   |
| x * y    | Умножение - возвращает произведение чисел x и y               |
| x / y    | Деление - возвращает частное от деления x на y                |
| x ** y   | Возведение в степень - возвращает x в степени y               |
| sqrt x   | Квадратный корень - возвращает квадратный корень из x         |
| x % y    | Остаток от деления - возвращает остаток от деления x на y     |
| 1 / x    | Обратное число - возвращает число, обратное x (1/x)           |
| M+       | Добавить в память - добавляет текущее значение в память       |
| M-       | Вычесть из памяти - вычитает текущее значение из памяти       |
| MR       | Восстановить из памяти - возвращает значение из памяти        |
