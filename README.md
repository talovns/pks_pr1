# Правила пользования калькулятором:
- Для запуска программы запустите файл Program.cs 
- После откроется терминал, в котором вы вводите свое выражение
- Обязательно разделять числа и знаки пробелами (например: 2 + 2, sqrt 4)
- sqrt - считается знаком и означает квадратный корень из числа (обязательно с маленькой буквы)
- Для получения ответа нажмите на клавиатуре ENTER
